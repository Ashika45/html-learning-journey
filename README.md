
# 🌐 HTML Learning Journey

I started this to build a strong foundation in web development and to stay consistent with my daily practice. Instead of just watching tutorials, I’m writing notes in my own words and practicing with small examples and mini projects.

## 📚 What I’m Learning

- Basic structure of HTML
- Headings, paragraphs, and formatting
- Lists (ordered and unordered)
- Links and images
- Forms and inputs
- Tables
- Semantic HTML (ongoing)

## 🧠 How I Learn

For each topic, I:
- Understand the concept
- Try it out with code
- Make small mistakes and fix them
- Write short notes to remember it better

## 📁 Structure

You’ll find:
